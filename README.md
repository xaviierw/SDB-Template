## Getting Started for Frontend

Install dependencies:

```bash
npm install
```

Start the frontend:

```bash
npm run dev
```

Open the local URL shown in the terminal:

```text
http://127.0.0.1:5173
```

## Useful Commands

```bash
npm run dev
npm run lint
npm run build
```

## Main Files

- `src/data/mockData.js` - edit markets, products, customer profile, and mock prices
- `src/pages` - app pages such as Login, Dashboard, Market, Trade, Profile
- `src/components` - shared layout and UI components
- `src/styles` - CSS split by purpose

## Project Notes

- All data is mocked. Please connect frontend to backend using your own API keys.
- Login and registration are frontend-only placeholders.
- PayPal login is currently mocked and should be connected through the Flask backend later.
- The Trade page is intentionally a placeholder so each team can build its own buying flow.
- Do not edit `dist` manually. It is generated by `npm run build`.



## Getting Started for Backend

Create and activate a virtual environment:

```bash
python -m venv venv
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Start the backend:

```bash
python run.py
```

The API should run at:

```text
http://127.0.0.1:5000
```

## Test Endpoint

Open this in your browser:

```text
http://127.0.0.1:5000/
```

Expected response:

```json
{
  "message": "Straits Digital Bank API is running"
}
```

## Project Notes

- Do not commit real API keys.
- Use a `.env` file later for PayPal and other Open Banking API credentials.
- PayPal login should be connected through the backend later, not directly from the frontend template.
- The current backend is only a starter and does not connect to the frontend yet.
